# SiriWave

A Swift Package library for creating Siri-like wave animations on iOS.

## Requirements

- iOS 15.0+
- Swift 6.1+

## Installation

### Swift Package Manager

Add SiriWave to your project through Xcode:

1. File â†’ Add Package Dependencies
2. Enter the package URL
3. Select version requirements

Or add it to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/your-username/SiriWave", from: "1.0.0")
]
```
